## Installation guide

1. Clone the repo
2. Run `aptos init`
3. Compile the code `aptos move compile`
4. Deploy the Move program on Aptos blockchain `aptos move publish`
5. Navigate to the client project `cd client_todo`
6. Run npm to install the dependencies `npm install`
7. Run `npm run dev`
